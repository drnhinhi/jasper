Jasper Retro Converter - Desenvolva Seus Próprios Estilos

Introdução

Para desenvolver seus próprios estilos para o Jasper Retro Converter, você precisará ter o programa instalado em seu computador. A seguir, fornecemos um exemplo de código de um arquivo de estilo.

Estrutura do Arquivo de Estilo

Um arquivo de estilo no Jasper Retro Converter deve ser salvo com a extensão .jrc. A estrutura básica do arquivo é a seguinte:

{ "name": "NomeDoEstilo", "palette": [ // Lista de cores em formato RGB ] }

Definindo o Nome do Estilo

A propriedade "name" define o nome do seu estilo. Substitua "NomeDoEstilo" pelo nome desejado. Exemplo:

{ "name": "VGA", }

Definindo as Cores da Paleta

A propriedade "palette" é uma lista de números que representam as cores em formato RGB. Cada cor é definida por três números consecutivos que representam os valores de Vermelho (R), Verde (G) e Azul (B). Exemplo:

{ "palette": [ 0, 0, 0, // Preto 33, 47, 73, // Azul Escuro 102, 45, 72, // Roxo Escuro 0, 104, 67, // Verde Escuro 150, 68, 52, // Marrom 80, 65, 57, // Cinza Escuro 178, 179, 183, // Cinza Claro 252, 233, 222, // Branco 255, 0, 90, // Vermelho 255, 154, 0, // Laranja 255, 220, 38, // Amarelo 0, 210, 50, // Verde 45, 160, 255, // Azul 120, 107, 140, // Roxo 255, 130, 150, // Rosa 255, 185, 140 // Bege ] }

Certifique-se de que os valores estejam em múltiplos de três, pois cada conjunto representa uma cor.

Exemplo Completo de Arquivo de Estilo

Salve o código abaixo em um arquivo com a extensão .jrc:

{ "name": "VGA", "palette": [ 0, 0, 0, 33, 47, 73, 102, 45, 72, 0, 104, 67, 150, 68, 52, 80, 65, 57, 178, 179, 183, 252, 233, 222, 255, 0, 90, 255, 154, 0, 255, 220, 38, 0, 210, 50, 45, 160, 255, 120, 107, 140, 255, 130, 150, 255, 185, 140 ] }

NÃO coloque comentarios no código se não não funciona!

Adicionando o Estilo no Jasper Retro Converter

  1. Abra o Jasper Retro Converter.
  2. Clique no botão "Carregar estilo .jrc" localizado na parte inferior direita da janela.
  3. Selecione o arquivo de estilo que você salvou.
  4. O novo estilo será adicionado à lista de estilos disponíveis para conversão.
